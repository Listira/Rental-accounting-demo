Pajak Penghasilan Badan (CIT)
Estimasi PPh Badan dari data pembukuan
Metode 1 — PPh Final UMKM 0,5% (PP 55/2022)
Berlaku jika omzet ≤ Rp 4,8 M/tahun dan masih dalam jangka waktu fasilitas PPh Final. Dihitung dari peredaran bruto (omzet), bukan laba.
Metode 2 — Tarif Umum Pasal 17 & 31E
Tarif 22% dari laba kena pajak. Fasilitas 31E: diskon 50% untuk bagian laba dari omzet s.d. Rp 4,8 M (berlaku jika omzet ≤ Rp 50 M).
Kredit Pajak & Penyesuaian
Kredit PPh 23: pajak yang sudah dipotong penyewa badan saat membayar sewa (2% dari bruto) — mengurangi PPh terutang metode tarif umum. Koreksi fiskal untuk biaya non-deductible dll (misal Biaya Parkir, entertainment tanpa daftar nominatif, dll), isi sesuai arahan konsultan pajak / akun yang dikoreksi di pembukuan.
Angsuran PPh 25 (Tahun Berikutnya)
Estimasi angsuran bulanan PPh 25 untuk tahun depan, dihitung dari PPh Badan terutang tahun ini dibagi 12 (dibulatkan ke atas) — sebagai acuan awal sebelum SPT Tahunan dilaporkan.
Data & Backup
Simpan data ke file di folder aplikasi — tidak bergantung pada cache browser
File Data Lokal (Auto-Save)
Hubungkan ke file data.json di folder yang sama dengan aplikasi ini (klik "Hubungkan / Buat File Data" lalu simpan di folder ini juga). Setelah terhubung, setiap perubahan (input penyewaan, jurnal, dll) otomatis tersimpan ke file data.json itu — bukan cuma di cache browser. Mau transfer ke komputer lain / kirim ke teman? Tinggal copy seluruh folder ini (berisi index.html + data.json), lalu di komputer tujuan buka index.html, klik "Buka File Data yang Sudah Ada" dan pilih data.json di dalam folder tadi. Browser akan mengingat koneksi ini — kalau lain kali aplikasi dibuka lagi dan koneksi sempat putus, cukup klik tombol "Hubungkan Lagi" yang muncul di bawah.

Tahun Buku
Aplikasi ini mencatat transaksi untuk satu tahun berjalan. Kalau tahun sudah berganti dan transaksi tahun ini sudah selesai diinput semua, klik "Tutup Tahun & Lanjut ke Tahun Berikutnya". Saldo akhir tahun ini otomatis jadi saldo awal tahun depan, dan laporan tahun depan bisa langsung dibandingkan (komparasi) dengan tahun ini. Data tahun yang sudah ditutup tetap tersimpan dan bisa dilihat lagi di sini — jangan khawatir, aman.
Backup Manual (JSON)
Download seluruh data sebagai file JSON untuk cadangan atau pindah komputer. Restore akan menimpa semua data yang ada sekarang.
Export & Import Data Operasional
Download data Armada, Penyewaan, Invoice, Bunga Leasing, Penyewa, dan Jurnal Umum sebagai satu file JSON terpisah — cocok untuk pindahan data atau dipakai lagi setelah Reset Total. Import akan menimpa data-data tersebut saja; Chart of Account, saldo awal, dan riwayat tahun yang sudah ditutup tidak terpengaruh.
Data Dummy (Testing)
Isi jurnal umum dengan transaksi dummy Januari–Desember (pendapatan, HPP, beban operasional, penyusutan, pelunasan piutang/hutang) — angka dibuat fluktuatif per bulan (mengikuti pola musiman rental kendaraan: ramai saat libur sekolah & akhir tahun, sepi awal tahun) supaya mendekati kondisi asli.
Opsi pertama mengisi riwayat tahun sebelumnya sebagai pembanding (dan menyesuaikan saldo awal tahun berjalan), tanpa mengubah jurnal tahun berjalan. Opsi kedua menghapus & mengisi ulang jurnal tahun berjalan, dan juga menambahkan contoh data Armada, Penyewa, Penyewaan, Invoice (variasi status lunas/belum bayar/jatuh tempo & PPN/diskon), dan Bunga Leasing supaya pengujian fitur operasional bisa komprehensif. Backup dulu kalau masih perlu data yang ada sekarang.
Reset / Hapus Semua Data
Reset ini khusus untuk tahun berjalan (tahun aktif yang sedang dipilih): menghapus seluruh data Penyewaan, Invoice, dan Jurnal Umum tahun berjalan (termasuk data dummy/testing) sehingga aplikasi kembali kosong untuk mulai input data yang sebenarnya. Status semua armada akan dikembalikan ke "tersedia". Data Armada itu sendiri tidak ikut terhapus. Data tahun-tahun sebelumnya yang sudah ditutup (riwayat saldo, laporan keuangan, jurnal archive) juga TIDAK ikut terhapus dan tetap aman. Chart of Account & saldo awal juga tidak ikut terhapus. Download backup dulu kalau masih perlu data yang ada sekarang. Tindakan ini tidak bisa dibatalkan.
Reset Total Aplikasi (Factory Reset)
Mengembalikan aplikasi ini 100% ke kondisi awal install / benar-benar kosong — Chart of Account & saldo awal kembali ke nilai bawaan, dan seluruh Armada, Penyewaan, Invoice, Bunga Leasing, Penyewa, Jurnal Umum, Pajak, Tahun Buku, Riwayat Tahun & arsip Jurnal, serta koneksi file lokal akan DIKOSONGKAN (tidak kembali ke data contoh — benar-benar kosong). Gunakan ini hanya jika ingin memulai dari nol sepenuhnya (misal mau dipakai untuk bisnis/cabang lain). Sebaiknya download dulu "Data Operasional" di atas (Armada, Penyewaan, Invoice, Leasing, Penyewa, Jurnal) dan/atau Backup Manual sebelum reset, supaya nanti tinggal di-import kembali. Tindakan ini TIDAK BISA DIBATALKAN.
